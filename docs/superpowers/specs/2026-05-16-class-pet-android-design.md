# 班级萌宠 Android App 设计文档

## 概述

一款面向小学班级管理的 Android 应用。老师在手机上操作，通过发放积分奖励驱动萌宠成长，大屏幕展示萌宠和班级积分信息。混合养成趣味性与班级行为激励机制。

**技术栈**：Kotlin + Jetpack Compose，Room 本地存储，Android Presentation API 投屏。

## 角色与设备

- **老师**：唯一操作者，使用 Android 手机
- **学生**：不直接操作 app，通过教室大屏幕观看萌宠状态
- **手机端**：操作面板（发放奖励、查看记录、管理萌宠）
- **大屏端**：展示界面（萌宠动画 + 班级积分信息面板）
- **投屏方式**：MediaRouter 检测设备 → Presentation API 驱动第二屏，非屏幕镜像

## 整体架构

单 Activity + Compose Navigation（底部三 Tab）+ Room 数据库。

```
MainActivity
  ├── Tab 1: 萌宠主页（宠物展示、改名、换外观）
  ├── Tab 2: 奖励发放（预设快捷按钮 + 自定义输入）
  ├── Tab 3: 奖励记录（时间倒序列表、支持撤销）
  └── Top Bar: 投屏控制（设备扫描、连接状态）

Presentation (大屏端)
  ├── 左侧 60%: 萌宠形象 + 名字 + 阶段
  └── 右侧 40%: 总积分 + 成长进度条 + 最近奖励
```

**状态管理**：MainViewModel + StateFlow，手机端和大屏端共享同一 ViewModel，数据变化两端同步更新。

## 萌宠成长系统

### 五个阶段

| 阶段 | 积分范围 | 视觉特征 |
|------|---------|---------|
| 蛋 (EGG) | 0–49 | 初始状态，蛋壳有裂纹动画 |
| 幼崽 (BABY) | 50–149 | 破壳而出，小巧可爱，走路不稳 |
| 少年 (TEEN) | 150–299 | 活泼好动，蹦跳、转圈等动作 |
| 成年 (ADULT) | 300–499 | 成熟形态，带有特殊配饰 |
| 成熟 (MATURE) | 500+ | 完全体，炫彩特效 |

### 成长机制

- **自动进化**：累计积分达到阈值自动触发，大屏播放进化动画（3-5 秒），手机端弹出提示
- **阶段内进度**：进度条显示当前阶段内的完成百分比，影响萌宠体型大小和外观细节
- **只升不降**：撤销奖励扣回积分后不会降级

### 进化特效

- 蛋 → 幼崽：破壳动画
- 幼崽 → 少年：光芒包裹成长
- 少年 → 成年：蜕变特效
- 成年 → 成熟：彩虹 + 星星粒子

## 奖励发放系统

### 发放界面（Tab 2）

- **快捷预设按钮**：一键发放，支持 6 个默认预设（举手回答 +2、作业优秀 +5、帮助同学 +3、卫生整洁 +2、阅读认真 +3、课堂专注 +4）
- **自定义输入**：手动输入分数 + 可选备注
- **预设管理**：长按编辑/删除，拖拽排序，最多 12 个预设，每个预设包含 label/icon/score/color/sortOrder
- **操作流程**：点击 → 震动反馈 → 飘 "+N" 动画 → 大屏同步动画 + 进度条推进 → 自动记录

### 奖励记录（Tab 3）

- 时间倒序列表，显示场景、分值、时间
- 左滑撤销最近一条记录，积分同步扣回
- 按日期筛选查看

## 萌宠自定义

### 首次启动向导

1. **选择类型**：小猫 / 小狗 / 小兔 / 小狐狸 / 熊猫 / 小恐龙 (6 种)
2. **选择主色调**：6 种预设颜色 (红/黄/绿/蓝/紫/粉) + 自定义 hex
3. **起名字**：限 6 个汉字以内

### 后续修改

Tab 1 萌宠主页提供"改名字"和"换外观"入口，随时可修改，大屏实时同步。

## 大屏展示界面

- **左侧 60%**：萌宠动画形象（当前阶段对应的外观）、名字、阶段名称
- **右侧 40%**：总积分大数字、成长进度条（距下一阶段还需多少分）、最近奖励滚动列表
- **动画同步**：加分时飘字动画、进度条平滑推进（300ms）、萌宠开心反应；进化时全屏特效

## 数据模型

### Pet（单行表）
| 字段 | 类型 | 说明 |
|------|------|------|
| name | String | 萌宠名字 |
| type | String | cat/dog/rabbit/fox/panda/dino |
| color | String | 主色调 hex |
| totalScore | Int | 累计总分 |
| stage | String | EGG/BABY/TEEN/ADULT/MATURE |

### RewardLog
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Long (PK) | 自增主键 |
| label | String | 奖励场景名 |
| score | Int | 分值 |
| note | String? | 可选备注 |
| timestamp | Long | 发放时间 |

### PresetReward
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Long (PK) | 自增主键 |
| label | String | 按钮文字 |
| icon | String | emoji 图标 |
| score | Int | 默认分数 |
| color | String | 按钮颜色 |
| sortOrder | Int | 排列顺序 |

## 边缘情况处理

- **首次启动**：数据库无 Pet 记录 → 进入设置向导，完成前不显示主界面
- **撤销奖励**：最近一条记录可左滑删除，扣回积分，已进化阶段不降级
- **投屏断开**：手机正常使用，Top Bar 显示未连接；重连后自动恢复大屏状态
- **无投屏设备**：投屏为可选功能，不影响手机端核心流程
- **数据备份**：纯本地存储，V2 可考虑 JSON 导出功能
